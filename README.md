# bilibiliSpider
抓取bilibili视频弹幕，生成词云

## 背景说明
通过爬虫获取指定av号的视频弹幕，分词之后生成词云  
*本项目所有代码在Python3.6.0下测试通过*

## 使用的模块
- requests(2.12.4)
- re
- matplotlib(2.0.0)
- wordcloud(1.3.1)
- jieba(0.38)
